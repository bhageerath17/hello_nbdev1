There are few clustering algorithms for which the number of cluster's is not pre-determined. Examples include DBSCAN and HDBSCAN

DBSCAN

DBSCAN doesn't need number of clusters a prior. The model depends on eps and min_samples. By setting eps=2 and min_samples=4, we get 6 clusters

Hyper-Parameter optimization

By selecting eps from the [0.1, 0.5, 1, 2] and min_samples from [3, 6, 10, 15, 30], the model responds in the following way.

  • The models are saved in the nbs/models/dbscan-* folder as .joblib files.
  • Cluster label for each product is saved in the nbs/models/dbscan-* folder as .parquet file
  • Counts of products in each cluster are saved in the nbs/models/dbscan-* folder as .csv file
